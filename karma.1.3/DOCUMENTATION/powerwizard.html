Powerwizard

Basic checkbox mass toggle plugin. This plugin support the use of the SHIFT key to select multiple checkboxes, can be used with requirejs and it can be used with CSS3 input styling.

Files To Include

Files that you have to include in your <head> section.

<script src="js/plugins/powerwizard.1.0.min.js"></script>
<link rel="stylesheet" href="css/powerwizard.css" />

How To Use

How to use HTML elements.

<ul class="powerwizard big-title">
    <li class="active">
        <h3>1. Step one</h3>
        <h4>With validation.</h4>
    </li>
    <li>
        <h3>2. Step two</h3>
        <h4>Grid and raster</h4>
    </li>
    <li>
        <h3>3. Step three</h3>
        <h4>Use tables</h4>
    </li>
</ul>
<div class="powerwizard-content">
    <div class="powerwizard-step">
        content goes here
    </div>
    <div class="powerwizard-step">
        content goes here
    </div>
    <div class="powerwizard-step">
        content goes here
    </div>
</div>

Plugin Usage

How to setup and use the plugin.

//init
$('.powerwizard').powerWizard({
    containerSelector:'.powerwizard-content',
    stepsSelector:'.powerwizard-step',
    nextButtonSelector:'.pw-next',
    prevButtonSelector:'.pw-prev',
    navButtonSelector:'.pw-nav',
    activeClass:'pw-active',
    disableClass:'pw-disable',
    clickableTitles: true,
    historyTitles: false,
    startAt: 0,
    useAjax:true,
    onNext:function(ui){
        //ui.step(element)
    },
    onPrev:function(ui){
        //ui.step(element)
    },
    onSwitch:function(ui){
        //ui.step(element)
    },
    onSubmit:function(ui){
        //ui.step(element)
        //ui.output(tring))
    },
    ajaxSuccess:function(ui){
        //ui.step(element)
    },
    ajaxError:function(ui){
        //ui.step(element)
    },
    onValidate:function(ui){
        //ui.step(element)
        
        // example: how to use validation
        // this will validate every input with the class 'required'
        
        var minChar     = 2;
        var returnFalse = false;
        
        $(ui.step).find('.required').each(function(){
            if($(this).is(':checkbox')){
                if($(this).is(':checked')){					
                    $(this).parents('div').removeClass('has-error');	
                }else{
                    $(this).parents('div').addClass('has-error');	
                    returnFalse = true;					
                }	
            }else{
                if($(this).val().length > minChar){					
                    $(this).parent().removeClass('has-error');	
                }else{
                    $(this).parent().addClass('has-error');	
                    returnFalse = true;					
                }	
            }
        });
        if(returnFalse === true){
            if(!$('.alert-wrap').length){
                $(ui.step).children('.inner-padding').prepend('
Error: One or more required fields are empty!
');
            }
            return false;
        }else{
            $(ui.step).find('.alert-wrap').remove();	
        }
        
        //this function works with a boolean value
        //return a false or true value to use this validation callback
        
    },
});

//public methods
$('.powerwizard').powerWizard('destroy');

Plugin Options

A list of plugin options.

Options Type Default Desctription Since
containerSelector string .powerwizard-content The container or wrapper of the steps. 1.0
stepsSelector string .powerwizard-step The steps that are inside the containerSelector. 1.0
nextButtonSelector string .pw-next The next button selector. 1.0
prevButtonSelector string .pw-prev The prev button selector. 1.0
navButtonSelector string .pw-nav The navigation selector. 1.0
activeClass string pw-active Active class used for the titles, navigation and buttons. 1.0
disableClass string pw-disable Disable class used for the titles, navigation and buttons. 1.0
clickableTitles boolean true Titles can be used as navigation. 1.0
historyTitles boolean false Show opened titles(active class will stay if you go back a step). 1.0
startAt integer 0 Start with a different step. Default = 0(first step). 1.0
useAjax boolean true Use AJAX to send the form. 1.0
onNext function function(ui){} Callback function that runs if you go to the next step. 1.0
onPrev function function(ui){} Callback function that runs if you go to the previous step. 1.0
onSwitch function function(ui){} Callback function that runs everytime you switch between a step. 1.0
onSubmit function function(ui){} Callback function that runs if a user submits the form. 1.0
ajaxSuccess function function(ui){} Callback function that runs if the form has been send with AJAX. 1.0
ajaxError function function(ui){} Callback function that runs if there's an error with the AJAX part. 1.0
onValidate function function(ui){} Callback function that can be used to validate a step. This function works with a boolean value, if there's an false value inside the function the wizard will stop. Example: function(){ return false ] will stop the wizard. 1.0

Globally And Per-Call Overridable Options

This plugin has per-call(as the example above shows) and globally overridable options. Globally options can be used to override the default options, this is best used if you want to override the plugin options by default, this will prevent any issues with the plugin, as you dont have to change any thing in the plugin core it self.

How to use: $.fn.pluginName.key = 'value';
