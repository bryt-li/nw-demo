Tinycontextmenu

A very basic contextmenu that only shows/hide the right click menu.

Files To Include

Files that you have to include in your <head> section.

<script src="js/plugins/tinycontextmenu.1.0.min.js"></script>
<link rel="stylesheet" href="css/tinycontextmenu.css" />

How To Use

How to use HTML elements.

<div class="tinycontextmenu" id="tcm-demo-1">
    <ul>
        <li><a href="#"><i class="fa fa-edit"></i> Edit</a></li>
        <li><a href="#"><i class="fa fa-cut"></i> Cut</a></li>
        <li><a href="#"><i class="fa fa-copy"></i> Copy</a></li>
        <li><a href="#"><i class="fa fa-paste"></i> Paste</a></li>
        <li class="seperator"></li>
        <li>
            <a href="#"><i class="fa fa-trash-o"></i> Delete <i class="fa fa-caret-right pull-right"></i></a>
            <ul>
                <li><a href="#"><i class="fa fa-file"></i> File</a></li>
                <li><a href="#"><i class="fa fa-folder"></i> Folder</a></li>
                <li><a href="#"><i class="fa fa-warning"></i> All files</a></li>
            </ul>
        </li>
        <li><a href="#"><i class="fa fa-folder-open"></i> Move to</a></li>
    </ul>
</div>

Plugin Usage

How to setup and use the plugin.

//init
$('#tiny-context-menu').tinyContextMenu({
    container: '#tcm-demo-1',
    offsetX:0,
    offsetY:0,
    onShow:function(ui){
        //ui.container(element)
    },
    onHide:function(ui){
        //ui.container(element)
    }
});
    
//public methods
$('#tiny-context-menu').tinyContextMenu('destroy');

Plugin Options

A list of plugin options.

Options Type Default Desctription Since
container string The container or wrapper of the context menu. 1.0
offsetX integer 0 Offset X position. 1.0
offsetY integer 0 Offset Y position. 1.0
onShow function function(ui){} Callback function that runs if the menu is triggerd(visible). 1.0
onHide function function(ui){} Callback function that runs if the menu is triggerd(hidden). 1.0

Plugin Callback Functions

Below a list of callback function parameters.

Function Parameter Returns Desctription Since
onShow ui.container element This will return the container. 1.0
onHide ui.container element This will return the container. 1.0

Globally And Per-Call Overridable Options

This plugin has per-call(as the example above shows) and globally overridable options. Globally options can be used to override the default options, this is best used if you want to override the plugin options by default, this will prevent any issues with the plugin, as you dont have to change any thing in the plugin core it self.

How to use: $.fn.pluginName.key = 'value';
