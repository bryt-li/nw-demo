Plugin Usage

How to setup and use the plugin.

//starts the loadingbar
$.nanoGress.start({
	target: '.targetclass'
});

//ends the loadingbar
$.nanoGress.end({
	delay: 2000,
	onEnd: function(){}
});

Plugin Options

A list of plugin options.

Options Type Default Desctription Since
target string body The target where the loadingbar should load. Default = body. This option is only used in the public method 'start'. 1.0
delay integer 0 Delay the ending of the loading bar. This option is only used in the public method 'end'. 1.0
onEnd function function(){} Callback function that runs if the plugin ends. This option is only used in the public method 'end'. 1.0

Globally And Per-Call Overridable Options

This plugin has per-call(as the example above shows) and globally overridable options. Globally options can be used to override the default options, this is best used if you want to override the plugin options by default, this will prevent any issues with the plugin, as you dont have to change any thing in the plugin core it self.
