Bootstrap

This theme is build with bootstrap, so we have used a couple of plugins(third-party) that have build to work with bootstrap. These plugins can be found inside the folder called 'bootstrap'.

Overview

A list of all third-party javascript/jQuery plugins.

  • Bootbox.js - http://bootboxjs.com/
  • Colorpicker - http://www.eyecon.ro/bootstrap-colorpicker/
  • Datepicker - http://www.eyecon.ro/bootstrap-datepicker/
  • Fileupload - http://jasny.github.io/bootstrap/javascript/
  • Holder - https://github.com/imsky/holder
  • Input mask - http://jasny.github.io/bootstrap/javascript/#inputmask
  • Select2 - http://ivaynberg.github.io/select2/
  • typeahead.js - http://twitter.github.io/typeahead.js/
