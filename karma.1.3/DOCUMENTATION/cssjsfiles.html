CSS/JS Files

An overview of the CSS/JS files.

About

We have put JS & CSS files into 3 groups, 1. custom, 2. thirdparty, 3. bootstrap.

1. custom files
These are custom made plugins just for this theme. Infomation about these plugins can befound inside this documentation.
2. thirdparty files
These are plugins not made by use. Infomation about these plugins can befound on external sites(plugin websites).
3. Bootstrap files
These are plugins not made by us and they are build to fit Bootstrap. This Bootstrap group includes both core and thirdpraty files. Infomation about these plugins can befound on external sites(plugin websites).

CSS files

Most CSS files dont need an explanation but there are a few that need one.

base.css
The main CSS file. Notice: as bootstrap CSS files are added inside the <head> the rest of the CSS files are imported inside this file.
boostrap-custom.css
This is the bootstrap reset CSS file, here you will find modified boostrap CSS code(inc bootstrap's third-party plugins). This way you dont have to worry that all is lost when you upgrade the Boostrap CSS core file.
boostrap-extended.css
This is the boostrap extended CSS file, here you will find stuff thats build on top of Bootstrap.
errorpages.css
The same as the base.css file but only used for the error pages.
helpers.css
Our small sort-of CSS framework, this file will provide sa couple of helper classes.
login.css
The same as the base.css file but only used for the login pages.
thirdparty-reset.css
This is the reset file for the third-party plugins. This way you dont have to worry that all is lost when you upgrade one of the plugins.

JS files

Most JS files dont need an explanation but there are a few that need one.

demo.js
Some extra jQuery run on start(index.html).
login.js
Plugins used for the login pages.
main.js
Custom made code.
plugins.js
Plugin settings can be found here.
