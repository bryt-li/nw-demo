Bookmark

An overview about how to bookmark this website in IOS(ipad/iphone), Windows(7/8), Blackberry(playbook) and Android. All PSD files are included to build your own icons and splash screens.

How To Bookmark

IOS webapp bookmark: http://lifehacker.com/5809338/add-web-site-bookmarks-to-your-iphones-homescreen

Windows 7 pin website: http://www.sitepoint.com/pinned-sites-with-internet-explorer-9-and-windows-7/

Windows 8 add tile: http://www.bleepingcomputer.com/tutorials/create-web-site-tile-in-windows-8/

Playbook browser shortcut: http://www.dummies.com/how-to/content/add-a-browser-shortcut-to-your-blackberry-playbook.html

Android add to homescreenhttp://www.maketecheasier.com/pin-bookmarks-to-home-screen-android/

Windows 8 tile

What we have used in our theme head section.

<!-- Tile name -->   
<meta name="application-name" content="Karma">
<!-- Tile background color --> 
<meta name="msapplication-TileColor" content="#333333" />
<!-- Tile icon image --> 
<meta name="msapplication-TileImage" content="images/mobile/windows8-icon.png" />

More info about this http://www.buildmypinnedsite.com/en
