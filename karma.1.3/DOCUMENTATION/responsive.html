Mediaqueries

Below are the mediaqueries that we have used for this theme, these mediaqueries can befound in the main css dir and its called mediaqueries.css.

Notice: bootstrap has its own mediaqueries and these can befound in the bootstrap.css file located in the bootstrap dir.

http://css-tricks.com/css-media-queries/

@media only screen and (min-width : 300px){ /* layout - no sidebar(s) visible */ }

@media only screen and (min-width : 768px){ /* layout - no sidebar(s) visible */ }

@media only screen and (min-width : 1000px){ /* layout - left sidebar(s) visible */ }

@media only screen and (min-width : 1200px){ /* layout - all sidebar(s) visible */ }

@media all and (max-width : 1000px){ /* timeline layout switch */ }

@media all and (max-width : 1200px){ /* rasterline(breakpoint-lg) horizontal flip */ }

@media all and (max-width : 992px){ /* rasterline(breakpoint-md) horizontal flip */ }

@media all and (max-width : 768px){{ /* rasterline(breakpoint-sm) horizontal flip */ }
